# Projects

Every conception item — feature project, incident, or document — lives here. One folder per item at `projects/YYYY-MM/YYYY-MM-DD-slug/`, with a `README.md` and optional `notes/` and `deliverables/` directories.

Kind is recorded in each README header:

- `**Kind**: project` — a new feature or planned behaviour change.
- `**Kind**: incident` — a bug, outage, or unexpected behaviour.
- `**Kind**: document` — a plan, report, investigation, or audit.

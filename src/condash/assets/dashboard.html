Repositories

One repo per line. Slashed names (myorg/backend) match depth-2 scan hits. Append submodules as reponame: sub/one, sub/two. /pr resolves the default base branch from origin/HEAD on the main checkout at call time; no per-repo override is stored here.

Open with

Each slot is a fallback chain of shell-style commands (one per line). The literal {path} is replaced with the repo's absolute path. The first command that starts is used.

PDF viewer

Fallback chain of shell-style commands tried for *.pdf files (one per line). The literal {path} is replaced with the file's absolute path; if the command has no {path}, the path is appended as the last argument (so evince behaves the same as evince {path}). Leave empty to use the OS default (xdg-open / open / startfile). Images and Markdown always use the OS default.
